# Segunda Entrega – Proyecto SmartPrice

## Descripción
La carpeta **2da Entrega** contiene la documentación y evidencias del avance del proyecto **SmartPrice**, un sistema diseñado para comparar precios entre supermercados y apoyar la toma de decisiones de los usuarios.  
Los entregables están organizados por secciones que abarcan competencias, diseño, proceso, producto, requisitos y presentación del avance.

## Objetivo del Proyecto
Desarrollar una plataforma que permita comparar precios de productos en supermercados para optimizar la toma de decisiones del consumidor, mediante una interfaz accesible, actualizada y centrada en el usuario.

---

## Metodología
El proyecto se desarrolla con metodología ágil basada en **sprints**, gestionado con **Git y GitHub**, documentación en **Markdown**, y retroalimentación continua con el cliente para validar avances.
