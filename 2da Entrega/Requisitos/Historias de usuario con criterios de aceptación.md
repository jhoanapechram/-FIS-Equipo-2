# Historias de usuario - Smart Price

## Tabla de Historias de Usuario

| #   | Requisito                       | Historia de Usuario                                                                 | Criterios de Aceptación                                                                                             | Excepciones                                                                                                                                                 |
|-----|---------------------------------|-------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| RF1 | Integración de APIs de Supermercados | Como comprador, quiero consultar los precios de un producto en distintos supermercados para elegir la opción más económica. | • Conexión estable con 5 supermercados<br>• 95% de consultas respondidas correctamente<br>• Actualización de precios cada 30 minutos<br>• Tiempo máximo de respuesta: 10 segundos | • Si una API no responde, se indicará "No disponible temporalmente"<br>• Si una API cambia su estructura, se desactiva mientras se actualiza<br>• Productos sin stock o descontinuados no aparecen en resultados |
| RF2 | Historial de Precios            | Como usuario frecuente, quiero ver la evolución de precios de un producto para decidir el mejor momento de compra. | • Historial mínimo de 12 meses<br>• Gráficas visuales de evolución<br>• Consulta del historial en menos de 2 segundos<br>• Respaldo automático diario | • Productos recientes mostrarán nota "Datos desde [fecha]"<br>• Si hay periodos sin datos, se muestran espacios marcados<br>• Registros mayores a 12 meses se archivan |
| RF3 | Comparación de Precios en Tiempo Real | Como comprador, quiero comparar precios en varias tiendas para ahorrar dinero. | • Comparación simultánea de 3 tiendas o más<br>• Indicador visual del precio más bajo<br>• Cálculo automático del precio por unidad<br>• Resultados en menos de 3 segundos | • Si un producto no existe en una tienda, se muestra "No disponible"<br>• Si hay menos de 3 tiendas disponibles, aparece advertencia "Solo disponible en tales tiendas"<br>• Productos sin unidad estándar no muestran precio unitario<br>• Si ninguna API responde, se muestra error general |
| RF4 | Predicción de Precios con IA    | Como usuario, quiero recibir alertas cuando un producto suba o baje de precio para aprovechar ofertas. | • Predicciones a 7, 14 y 30 días<br>• Precisión mínima del 85%<br>• Gráficas con nivel de confianza<br>• Actualización automática semanal | • Productos con poco historial mostrarán "Datos insuficientes"<br>• Si la variación es mayor al 50%, se indica "Predicción poco confiable"<br>• En eventos extraordinarios, se pausarán predicciones con aviso<br>• Alertas solo para usuarios registrados con productos seguidos |
| RF5 | Validación de Confiabilidad de Ofertas | Como consumidor, quiero saber si una oferta es confiable o solo temporal. | • Clasificación en niveles (ALTA, MEDIA, BAJA)<br>• Indicador de color según nivel<br>• Estimación de la duración<br>• Cálculo del descuento real vs promedio | • Ofertas sin historial previo se marcan como "Nueva oferta - Sin validación"<br>• Productos con menos de 3 ofertas mostrarán "Datos insuficientes"<br>• Descuentos mayores al 70% llevan advertencia "Verificar con tienda" |
| RF6 | Geolocalización y Recomendación Inteligente | Como cliente, quiero que el sistema sugiera la tienda más cercana con buen precio para ahorrar tiempo y transporte. | • Solicitud de ubicación con consentimiento<br>• Precisión en distancia de 500 metros<br>• 3 modos de búsqueda<br>• Mapa interactivo con marcadores<br>• Estimación de tiempo de traslado | • Si el usuario rechaza ubicación, se solicita dirección manual<br>• Sin ubicación, se ocultan distancias y mapas<br>• Si no hay tiendas en 50 km, se muestra "No hay tiendas cercanas"<br>• El cálculo no considera tráfico en tiempo real<br>• Disponible solo en Mérida, posteriormente a México |
| RF7 | Asistente Virtual Inteligente (Chatbot) | Como usuario, quiero preguntar por productos en un chat para resolver dudas fácilmente. | • Respuesta en menos de 5 segundos<br>• Comprensión de lenguaje natural en español<br>• Mantiene contexto en mínimo 3 mensajes<br>• Menciona fuente de información | • Solo responde sobre precios, productos o compras<br>• Si la IA no está disponible, se muestra formulario de contacto<br>• Conversaciones limitadas a 20 mensajes<br>• Lenguaje ofensivo cierra la conversación<br>• No puede realizar compras |
| RF8 | Validación con Datos de Profeco | Como consumidor, quiero validar precios con información oficial de Profeco. | • Sincronización semanal con datos Profeco<br>• Mínimo 1,000 productos validados<br>• Coincidencia mínima del 85% en nombres<br>• Sello "Validado por Profeco" | • Si Profeco no tiene datos, no se muestra sello<br>• Si el sitio no responde, se usan datos anteriores con nota<br>• Productos fuera del programa no se validan<br>• Diferencias mayores al 10% se revisan manualmente |
| RF9 | Gestión de Usuarios y Preferencias | Registro y configuración de preferencias personalizadas. | • Registro con email y contraseña segura<br>• Verificación por código<br>• Selección de supermercados favoritos<br>• Guardado de ubicación y alertas<br>• Preferencias persistentes entre sesiones | • Si el email ya existe, se notifica<br>• Si la contraseña es débil, se muestran ejemplos válidos<br>• Si el código expira, se permite reenviar<br>• Si no elige supermercados, se aplican 3 por defecto |
| RF10| Sistema de Notificaciones       | Envío de alertas sobre cambios de precio y ofertas. | • Notificación push con baja del 5% o más<br>• Email semanal con resumen<br>• Configuración de umbral personalizado<br>• Límite de notificaciones diario ajustable | • Si el usuario desactiva notificaciones, solo email<br>• Reintento hasta 3 veces en caso de fallo<br>• Cambios menores se agrupan en el resumen |
| RF11| Gestión de Listas de Compra     | Creación y comparación de listas en distintos supermercados. | • Crear lista con nombre y productos<br>• Agregar productos por búsqueda, escaneo o voz<br>• Comparación de costo total<br>• Exportación en PDF o link | • Si un producto no existe, se puede agregar manualmente<br>• Si no hay precio, se usa el último conocido con advertencia |
| RF12| Reporte de Discrepancias        | Reporte de precios incorrectos y seguimiento del estado. | • Botón "Reportar precio" en cada producto<br>• Formulario con datos clave<br>• Seguimiento del reporte<br>• Notificaciones de actualización | • Si no hay internet, se guarda y envía después<br>• Fotos opcionales si hay registro<br>• Reportes duplicados se consolidan |
