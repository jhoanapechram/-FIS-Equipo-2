# Requisitos – Proyecto SmartPrice

---

## Descripción general

La carpeta **Requisitos** reúne toda la documentación relacionada con la definición, análisis y organización de los requerimientos del sistema **SmartPrice**, incluyendo casos de uso, historias de usuario, priorización y el diagrama general del sistema.

## Objetivo de la carpeta

El propósito de esta carpeta es centralizar y documentar todos los requisitos del sistema para guiar el diseño, desarrollo y validación del prototipo de **SmartPrice**.
