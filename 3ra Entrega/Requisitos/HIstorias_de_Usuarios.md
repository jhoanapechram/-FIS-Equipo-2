#  Historias de Usuario-Smart Price
----
Tabla de Historias de Usuario
----
| Código | Requisito | Historia de usuario | Criterios de aceptación |
|---------|------------|---------------------|--------------------------|
| RF1 | Integración de APIs de Supermercados | Como usuario quiero consultar precios actualizados de múltiples supermercados automáticamente para encontrar la mejor opción sin visitar cada tienda físicamente. | • Al buscar un producto, se muestran precios de 5 supermercados diferentes.<br>• Cada resultado muestra: precio actual, disponibilidad en tienda y distancia.<br>• Indicador visual cuando los datos están siendo actualizados. |
| RF2 | Historial de Precios | Como usuario que planifica compras quiero ver la evolución histórica de precios de productos para identificar patrones y decidir el mejor momento de compra. | • Gráfica interactiva que muestra variación de precios últimos 12 meses.<br>• Línea de tendencia clara con puntos de cambios significativos.<br>• Marcadores que indican eventos especiales (ofertas, temporadas). |
| RF3 | Comparación de Precios en Tiempo Real | Como usuario, quiero comparar el precio de un producto en diferentes supermercados para escoger el mejor lugar en donde comprar. | • Comparación simultánea de tres o más tiendas diferentes.<br>• Indicador visual del precio más bajo. |
| RF4 | Predicción de Precios con IA | Como usuario, quiero un sistema que pueda predecir cuándo un producto baja o sube de precio para aprovechar todas las ofertas. | • Predicciones a siete, catorce y treinta días.<br>• Precisión mínima del 85%.<br>• Gráficas con nivel de confianza visible. |
| RF5 | Validación de Confiabilidad de Ofertas | Como usuario, quiero saber si una oferta es confiable para tomar decisiones de compra informadas. | • Clasificación de confianza en niveles: ALTA, MEDIA, BAJA.<br>• Indicador de color según el nivel de confianza (verde/amarillo/rojo).<br>• Estimación de la duración de la oferta.<br>• Cálculo de la oferta real vs el promedio histórico. |
| RF6 | Geolocalización y Recomendación Inteligente | Como usuario quiero que el sistema sugiera la tienda más cercana y con mejores precios para ahorrar tiempo y transporte. | • Solicitar ubicación con consentimiento explícito.<br>• Precisión de distancia de 500 metros.<br>• Tres modos de búsqueda: “Mejor Precio”, “Más Cercano”, “Balance”.<br>• Mapa interactivo con marcadores de tiendas.<br>• Estimación del tiempo de traslado. |
| RF7 | Asistente Virtual Inteligente (Chatbot) | Como usuario, quiero interactuar con un asistente virtual inteligente que entienda mis preguntas y me recomiende productos para obtener información fácilmente. | • Chatbot con comprensión de lenguaje natural.<br>• Reconocimiento de al menos 10 intenciones (ofertas, ubicación, comparación, historial, etc.).<br>• Respuestas con formato estructurado (texto, enlace, imagen o botón).<br>• Capacidad para mantener el contexto durante la conversación.<br>• Respuestas basadas en los datos reales del sistema. |
| RF8 | Validación con Datos de Profeco | Como usuario, quiero que los productos estén validados por la base pública de Profeco para asegurarme de que los precios sean confiables. | • Conexión semanal con la base pública “Quién es Quién en los Precios”.<br>• Sello visual de validación en los productos coincidentes.<br>• Opción de filtrar productos por “Validados por Profeco”.<br>• Registro automático de fecha y fuente de validación. |
| RF9 | Gestión de Usuarios y Preferencias | Como usuario, quiero poder crear una cuenta y configurar mis preferencias (supermercados favoritos, ubicación, alertas) para personalizar mi experiencia. | • Registro e inicio de sesión con autenticación de correo electrónico.<br>• Configuración de supermercados favoritos y productos seguidos.<br>• Activación y desactivación de alertas personalizadas.<br>• Edición o eliminación de preferencias guardadas.<br>• Almacenamiento persistente de datos de usuario. |
| RF10 | Sistema de Notificaciones | Como usuario, quiero recibir notificaciones cuando bajen los precios de los productos que sigo o alcancen mi precio objetivo para aprovechar las ofertas. | • Activación manual de notificaciones por producto.<br>• Envío de alerta cuando el precio disminuye un 5% o más.<br>• Incluye información de precio anterior, nuevo y tienda.<br>• Notificaciones disponibles por correo o dentro de la app.<br>• Registro histórico de las notificaciones enviadas. |
| RF11 | Gestión de Listas de Compra | Como usuario, quiero crear y gestionar listas de compra para comparar el costo total entre supermercados y optimizar mi presupuesto. | • Creación, edición y eliminación de listas de compra.<br>• Cálculo automático del total por supermercado.<br>• Cálculo del ahorro potencial entre tiendas.<br>• Indicación visual de productos no disponibles.<br>• Almacenamiento de listas en el perfil del usuario. |
| RF12 | Reporte de Discrepancias | Como usuario, quiero poder reportar precios incorrectos con evidencia para ayudar a mejorar la precisión del sistema y ayudar a otros usuarios. | • Formulario de reporte con campos: producto, tienda, precio real y comentario.<br>• Opción de adjuntar evidencia fotográfica.<br>• Mensaje visual de confirmación de envío.<br>• Registro automático de fecha y usuario remitente.<br>• Acceso administrativo para revisión y cierre de reportes. |
