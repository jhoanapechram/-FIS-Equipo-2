# Diseño – Proyecto SmartPrice

---

## Descripción general

La carpeta **Diseño** contiene la documentación relacionada con la creación, estructura, lineamientos y evolución del diseño del sistema SmartPrice.

Centralizar la documentación de diseño para asegurar coherencia visual y funcional entre los requerimientos, el prototipo y la implementación final.
