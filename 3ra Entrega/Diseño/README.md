# Diseño

Esta carpeta contiene los documentos relacionados con el diseño del prototipo de **SmartPrice**, incluyendo cambios visuales, correspondencia con los requisitos y la guía completa de diseño utilizada durante el proyecto.

## Contenido

### 1. [Cambios del Prototipo](https://github.com/jhoanapechram/-FIS-Equipo-2/blob/TERCERA-ENTREGA-V2/3ra%20Entrega/Dise%C3%B1o/Cambios_prototipo.md)
Documento que describe los ajustes solicitados y realizados en el prototipo, incluyendo historial de precios, predicciones, perfil del usuario, gestión de listas de compra y mensajes de confirmación.

### 2. [Correspondencia de Requerimientos](https://github.com/jhoanapechram/-FIS-Equipo-2/blob/TERCERA-ENTREGA-V2/3ra%20Entrega/Dise%C3%B1o/Correspondencia_Requerimientos.md)
Tabla donde se relacionan los **requisitos funcionales (RF1–RF12)** con las secciones del prototipo donde dichos requisitos están representados.

### 3. [Guía de Diseño](https://github.com/jhoanapechram/-FIS-Equipo-2/blob/TERCERA-ENTREGA-V2/3ra%20Entrega/Dise%C3%B1o/Gu%C3%ADa_Dise%C3%B1o.md)
Incluye el prompt utilizado para generar el diseño con IA y la guía completa de estilo: colores, tipografía, componentes UI, interacciones, responsividad y lineamientos visuales del proyecto.
