# Cambios Solicitados al Prototipo de SmartPrice

A continuación se describen los ajustes y mejoras solicitadas para el prototipo de SmartPrice. Estos cambios abarcan funcionalidades de historial de precios, predicciones, perfil del usuario, gestión de listas y mensajes de confirmación dentro del sistema.

## Cambios Realizados

- El usuario hace clic en el botón **"Ver Historial de Precios"** y se despliega la gráfica correspondiente.

- El sistema muestra una predicción (por ejemplo, “Se prevé una bajada de precios la próxima semana”) o programa una alerta automática cuando se presiona el botón **"Seguir producto"**.

- **En el apartado de Perfil:**
  - Si el usuario selecciona **Registrarse**, se muestra un formulario con campos de correo y contraseña.
  - Si selecciona **Editar Perfil**, el sistema despliega sus preferencias actuales.
  - El usuario puede modificar supermercados favoritos, ubicación, canal de notificaciones y radio de búsqueda.
  - El usuario presiona **“Guardar cambios”**.
  - El sistema muestra el mensaje de confirmación: “Cambios guardados con éxito”.

- **En la sección Carrito de la barra de navegación:**
  - El sistema muestra la pantalla **“Resumen de compra”** con las opciones **Crear nueva lista** o **Ver listas existentes**.
  - El usuario puede crear una nueva lista o editar una existente.
  - El usuario agrega productos desde la barra de búsqueda o desde su historial/favoritos.
  - El usuario puede eliminar o agregar productos, renombrar la lista o cambiar el orden.
  - Al presionar **“Guardar lista”**, el sistema guarda los cambios y muestra el mensaje “Lista guardada correctamente”, además de ofrecer la opción de exportarla.
  - El sistema recalcula el ahorro potencial y lo muestra en pantalla.

- Cuando se reporte un precio exitosamente, el sistema debe mostrar el mensaje:  
  **“Gracias por tu reporte, será revisado por el equipo de SmartPrice”.**


## 5. Mensaje al reportar un precio
